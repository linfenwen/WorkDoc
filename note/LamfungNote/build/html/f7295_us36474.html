FQA¶

  1. Where is the mc.exe
    • Win10
    • C:\Program Files (x86)\Windows Kits\8.1\bin\x64\mc.exe
  2. To compile the message text file, use the following command:
    • mc -U provider.mc
  3. To compile the resources that the message compiler generated, use the following command:
    • rc provider.rc
